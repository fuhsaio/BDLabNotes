# Overfitting 過度擬合
> 過度學習訓練資料，變得無法順利去預測或分辨 不是在訓練資料內的其他資料

</br>

### Training、Test errors 訓練、測試誤差
### Generalization errors 泛化誤差
  > 真實情況下模型的誤差

</br>

### 未擬合 underfitting
* model 太過簡單
* 訓練集 資料不具代表性、規模太小
* 分類節點(node)增加，錯誤率反而提高
### 過擬合 overfitting
* model 太過複雜
* 訓練集 錯誤率降低、但測試集 錯誤率提高
* 多重比較規則 Multiple Comparison Procedure
